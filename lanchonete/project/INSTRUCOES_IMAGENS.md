# Como Usar Imagens Locais no Sistema

Este guia explica como adicionar imagens aos produtos utilizando arquivos guardados no próprio computador.

## 1. Onde Guardar as Imagens
1. Abra a pasta do projeto do sistema.
2. Procure pela pasta chamada `ImagensProdutos` (ela fica dentro de `project`). Se a pasta ainda não existir, crie-a com exatamente esse nome.
3. Copie todas as fotos dos seus produtos para dentro desta pasta.
   - **Dica:** Tente usar nomes simples e sem espaços para os arquivos (ex: `hamburguer.jpg`, `coca-cola.png`).

## 2. Como Configurar no Sistema (Área Admin)
1. Acesse o painel de **Produtos** no sistema.
2. Clique em **Editar** em um produto ou em **Novo produto**.
3. No campo **"Nome do arquivo da imagem"**, digite exatamente o nome do arquivo que você colocou na pasta `ImagensProdutos`.
   - Exemplo: Se o arquivo se chama `x-tudo.jpg`, digite `x-tudo.jpg` no campo.
4. Clique em Salvar.

## 3. Observações Importantes
- Para que as imagens apareçam corretamente, o sistema (backend) precisa estar rodando no mesmo computador onde a pasta `ImagensProdutos` está.
- O sistema ajusta automaticamente as imagens para que caibam corretamente e de forma responsiva na interface de pedidos e no cardápio.
